```python
from konlpy.tag import Okt
okt = Okt()
okt.pos('그가 밥먹는 동안 내가 구조물의 내구성을 파악했다')
```


이렇게 트위터 기반의 오픈소스 한글 형태소 분석기인 Okt를 이용하면,

```
[('그', 'Noun'),
 ('가', 'Josa'),
 ('밥', 'Noun'),
 ('먹는', 'Verb'),
 ('동안', 'Noun'),
 ('내', 'Noun'),
 ('가', 'Josa'),
 ('구조', 'Noun'),
 ('물의', 'Noun'),
 ('내', 'Determiner'),
 ('구성', 'Noun'),
 ('을', 'Josa'),
 ('파악', 'Noun'),
 ('했다', 'Verb')]
```
