tensorflow를 이용하다보면, 아래와 같은 에러를 종종 마주한다.


```
ValueError: Failed to convert a NumPy array to a Tensor (Unsupported object type int) in Python
```


그럴때 이런 코드로 고쳐보자


```python
# int인 X에 대해,
X=np.asarray(X).astype(np.int)

# float인 y에 대해,
y=np.asarray(y).astype(np.int)
```
